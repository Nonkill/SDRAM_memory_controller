**RELEASE 1.0.1**

Контроллер памяти позволяет работать с памятью 42-45S83200J-16160J от Micron в режиме одиночного чтения и записи.

Имеется интерфейс для работы:

ADR_IN - для передачи адреса строки/столбца

BDR_IN - для передачи номер банка

DIN - для передачи записываемых данных

DOUT - для получения читаемых данных

RE_IN - режим чтения

WE_IN - режим записи

RDY - готовность данных

NRST - асинхронный (по его отрицательному фронту) сброс контроллера

CLK - тактовый сигнал

Полный цикл чтения или записи занимает - 11 циклов (было 9 циклов), так как необходимо соблюсти ограничения памяти по .

Были добавлены буферы на порты передач/приёма данных от памяти (DIN, DOUT).

Создан оберточный top-модуль.

Исправлен баг со сверхкоротким чтением из памяти и с неработающим RDY, добавлено условие, по которому сигнал будет подаваться только при реальном окончании чтении/записи.
